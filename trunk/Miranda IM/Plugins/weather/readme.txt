Buuf Weather Avatars

These Avatarset base of the Iconset by mattahan (get it HERE: http://www.deviantart.com/deviation/37966044/), big thanks for that! If you want redistribute this set dont forget the link to mattahan's Buuficons!



requirements:

Weather Protocol: http://addons.miranda-im.org/details.php?action=viewfile&id=2322
Avatar service plugin: http://addons.miranda-im.org/details.php?action=viewfile&id=2361 (Ansi) - http://addons.miranda-im.org/details.php?action=viewfile&id=2990 (Unicode)

clist_nicer: http://addons.miranda-im.org/details.php?action=viewfile&id=2602 (Ansi) - http://addons.miranda-im.org/details.php?action=viewfile&id=2601 (Unicode)
or
clist_modern: http://addons.miranda-im.org/details.php?action=viewfile&id=2996 (Ansi) - http://addons.miranda-im.org/details.php?action=viewfile&id=2103 (unicode)


installation notes:

-put images in plugins/weather folder.
-make sure weather protocol is marked as active in customize->contact pictures (do not specify a path to image for it) and avatars are enabled in your clist options.
-restart miranda
if images aren't shown reset them from contact menu->contact picture. If this won't help feel free to express your fillings in http://forums.miranda-im.org/showthread.php?t=4425
